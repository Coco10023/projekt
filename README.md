# WeatherTrip

WeatherTrip är en webbapp som låter användaren söka efter en ort och få:
- platsinformation
- aktuell väderdata
- 3-dygnsprognos
- karta
- aktivitetsförslag

## Tekniker
- HTML
- SCSS
- JavaScript
- Vite
- Leaflet
- Open-Meteo API
- GitHub Actions
- GitHub Pages

## Installation
npm install
npm run dev

## Build
npm run build
