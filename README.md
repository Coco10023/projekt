# WeatherTrip: https://coco10023.github.io/projekt/ 

## Rapport
Rapporten finns i projektets rot: `Projektuppgift Frontendutveckling Francisco`

WeatherTrip är en webbapp som låter användaren söka efter en ort och få:
- platsinformation
- aktuell väderdata
- 3-dygnsprognos
- karta
- aktivitetsförslag

## Tekniker
- HTML
- SCSS
- JavaScript
- Vite
- Leaflet
- Open-Meteo API
- GitHub Actions
- GitHub Pages

## Installation
npm install
npm run dev

## Build
npm run build
